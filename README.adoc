== Building

Each sample contains both a Maven (`pom.xml`) and Gradle (`build.gradle`) build.
The source code is shared — only the build descriptor differs.

[source,bash]
----
# Maven
./mvnw clean install

# Gradle
./gradlew build
----

== Requirements

* Java 17+
* Maven 3.9+ or Gradle 8+
